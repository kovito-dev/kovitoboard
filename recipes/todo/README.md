# TODO 管理

シンプルな TODO リストです。タスクの追加・完了・削除ができます。

## できること

- タスクの追加（Enter または「追加」ボタン）
- 完了の切り替え（チェックボックス）
- 削除（各行の「削除」ボタン）
- 最大 100 件まで保存

## 使う権限

- `own-data`: このレシピ専用のローカル保存領域

> プロジェクトのファイルや他のレシピのデータには一切触れません。

## データの保存場所

`app/data/todo/` 配下に KV ストアとして保存されます。
KB をアンインストールするとこのデータも消えます。

## 拡張のヒント

- 配色を変えたい: `pages/TodoPage.tsx` 内の Tailwind クラスを編集
- 期限管理を追加したい: `Task` 型に `dueDate` フィールドを追加
- カテゴリ分けを追加したい: `Task` 型に `category` を追加し、入力 UI を拡張

> 秘書エージェント（さくら等）に「TODO の背景色を青系に変えて」のように頼むと、対応してくれます。
