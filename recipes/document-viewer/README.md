# ドキュメントビュアー

プロジェクト内の Markdown（`.md`）ファイルを一覧表示・閲覧します。

## できること

- プロジェクト直下〜1 階層下までの `.md` ファイルを自動検出
- ファイルを選択すると本文を表示
- 「リロード」ボタンで一覧を再取得

## 使う権限

- `project-read`: プロジェクトファイルの読み取り（書き込みはしません）

> `.env` や `.git/` など、機密に該当するファイルは KB が自動で隠します。

## 拡張のヒント

- 表示する拡張子を増やしたい: `pages/DocumentViewer.tsx` の `EXTENSIONS` を編集
- レイアウトを変えたい: 同ファイルの JSX 構造を編集

> Kovito のコンシェルジュ「コビー」に「`.ts` も見られるようにして」と頼むと自動で対応します。
